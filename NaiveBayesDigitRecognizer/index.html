Statistical Data Analysis/Learning Algorithms

My recent research involved theoretical study of impure/disordered material using high energy physics tool the so called "holography", where I performed extensive statistical data analysis; material are impure in "random" ways, but often the underlying distribution function is known or there are good guesses. In particular, I had to employ clustering learning algorithm for data compression among other things (basic inferential data analysis, in particular hypothesis testing).

Knowledge projects:

Inspired by my research needs and the Machine Learning course, I developed the following Java application and deployed them successfully:

  1. A Java application which implements K-means unsupervised learning algorithm, in the context of an image color quantizer. The Java linear algebra library EJML (Efficient Java Matrix Library) was used. The code does not use any of the pre-existing Machine Learning libraries such as scikit-learn. The application comes with a simple Java Swing GUI (Graphical User interface). The final product was successfully deployed two forms:
    • Java Web Start Technology. Launch my application from this page. For troubleshooting see instructions here.
    • Using appbundler utility, I packaged the jars into a native Mac application which can be launched from end user's machine.
    • Documentation can be found here.
    • Find the source codes on GitHub.
  2. Description of the final project of the Hadoop/MapReduce course: The data set was an anonymized sample taken from "Udacity forum". Our task was to write scripts in Python (mappers and reducers) to get insight into the forum posts. There were multiple goals:
    • To write mappper/reducer scripts to find if there is correlation between the length of the forum "questions" and their corresponding "answer(s)".
    • Every entry of the forum had a "tagnames" filed (a collection of tags). Write mapper/reducer scripts to find the top-10 popular tags.
    • Write MapReduce code to find the most active hour (on the forum) of each student in the forum. If there are more than one active hour, output them all.
    • Here you can find MapReduce scripts I wrote in Python for the final project (successfully passed).
